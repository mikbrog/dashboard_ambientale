# Dashboard ambientale statica

## Descrizione

Sito per una dashboard ambientale che recupera dati da un file locale.
Può essere visualizzato anche online [qui](https://mikbrog.github.io/dashboard_ambientale/)

## Funzionalità principali

- Pagina index.html di benvenuto per gli utenti.
- Pagina dati.html dove viene visualizzata la dashboard con filtri e barra di ricerca per i dati.
- Pagina spegazione.html contenente breve descrizione del progetto e l'autore.

## Tecnologie usate

- Python per avviare server per il sito
- Git per il versionamento del progetto
- Github per repository sul cloud
- Browser Google Chrome
- HTML,CSS e Javascript per la realizzaione del sito
- Framework Bootstrap per la realizzazione del sito

## Requisiti
- Git
- Python 3.13.13 o superiore
- Google Chrome Versione 148.0.7778.179 o superiore

## Istruzioni rapide

1. Scaricare la repository del progetto
2. Avviare server python con il comando ```python -m http.server 8000```
3. Connetersi al sito con l'url ```http://localhost:8000```
