# Istruzioni per l'installazione

## Requisiti

- Avere installato git sul proprio dispositivo
- Avere installato Google Chrome con versione 148.0.7778.179 o superiore
- Avere installato Python versione 3.13.13 o superiore

## Step 1 - Download Repository

Aprire il terminale edeseguire il comando
```bash
git clone https://github.com/mikbrog/dashboard_ambientale.git
```

## Step 2 - Avvio Server Python

Seguendo lo step precedente esegurie il comando
```bash
python -m http.server 8000
```

## Step 3 - Aprire il sito

Aprire chrome e digitare nella barra dell' url
```text
http://localhost:8000
```
e premere invio.
