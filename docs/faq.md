# F.A.Q.

### Come aprire il sito?

Dopo aver avviato il server locale python con il comando ``` python -m http.server 8000 ``` aprire l'url ```http://localhost:8000```

### Perché i dati non compaiono?

Assicurarsi che nella repository sia presente il file data.json

### Come scarico la repository?

Seguire le istruzioni nel documento per l'[installazione](installazione.md)

### Che browser utilizzare per aprire il sito?

Consigliato usare Google Chrome Versione 148.0.7778.179 o superiore poichè è stato testato solo con quello. L'utilizzo di altri browser potrebbe dare problemi di visualizzazione del sito.

### Ho trovato un bug nel progetto come lo segnalo?

Aprire un issues nella repository github del progetto.
